# Conference

> Two missions: (1) Teach anyone to build portfolios with free AI tools, (2) Connect volunteers with organizations needing low-cost database migration

## Sub-Projects

| Repo | Purpose |
|---|---|
| [conference-portfolio-builder](https://github.com/Momoiroart/conference-portfolio-builder) | Educational templates & tutorials |
| [conference-volunteer-platform](https://github.com/Momoiroart/conference-volunteer-platform) | AI migration platform for orgs |
| [conference-learn-guide](https://github.com/Momoiroart/conference-learn-guide) | Interactive tutorials |

## Our Goals

### Goal 1: Educational Platform
Teach anyone (kids, beginners, adults) how to use free AI tools (Ollama + Claude Code) to build portfolios, games, and cool projects.

### Goal 2: Volunteer Platform
Connect volunteers with organizations/schools that need low-cost database migration using agentic AI.

## Tech Stack (All Free!)

- **Ollama** - Free local LLM
- **Claude Code** - Free AI coding assistant
- **Astro** - Web framework
- **Vercel/Netlify** - Free hosting
